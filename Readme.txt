Cách chạy server
B1: Mở terminal
B2: Chạy lệnh node index.js
 node index.js
B3: Chạy http://localhost:3000/

Cách tạo server
B1: Tải Nodejs <https://nodejs.org/en>
B2: Chạy terminal
 npm init ( Khởi tạo project nodejs với tên project (VD:TestIOT)) 
 npm i express (Tải thư viện express)
 npm i ws (Tải thư viện websocket)
B3: Chạy server 
node index.js
